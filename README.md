# 🖊️ 中文小说写作技能包 (Novel Skill)

一套基于 Claude Code 的中文小说写作技能系统，支持多种类型小说的全流程创作。

## ✨ 特性

- 🎭 **多类型支持** — 言情、玄幻、悬疑、科幻、历史、都市等
- 📚 **全流程覆盖** — 从构思→大纲→人设→章节→润色→检查
- 🔍 **一致性管理** — 世界观、角色、伏笔、时间线自动追踪
- 🎨 **智能适配风格** — 根据类型自动切换写作风格
- 💬 **选择式交互** — 大模型给出选项，用户选择+自由输入

## 📦 技能列表

| 技能 | 命令 | 功能 |
|------|------|------|
| 项目创建 | `/project:小说-新建` | 创建新小说项目，初始化目录结构 |
| 世界观设定 | `/project:小说-世界观` | 构建世界观体系（地理/历史/规则） |
| 角色人设 | `/project:小说-人设` | 创建角色档案（性格/背景/关系/弧光） |
| 分层大纲 | `/project:小说-大纲` | 生成卷→章→场景→节拍的多层级大纲 |
| 伏笔管理 | `/project:小说-伏笔` | 伏笔埋设、追踪、回收管理 |
| 章节撰写 | `/project:小说-章节` | 按章节撰写正文，自动加载上下文 |
| 续写扩写 | `/project:小说-续写` | 续写/扩写/改写/缩减已有内容 |
| 文笔润色 | `/project:小说-润色` | 多维度润色优化，前后对比 |
| 一致性检查 | `/project:小说-检查` | 角色/世界观/时间线/伏笔逻辑校验 |
| 状态管理 | `/project:小说-状态` | 角色状态、时间线、已知信息追踪 |

## 📌 前置条件

- 安装并登录 **Claude Code**（支持 CLI / 桌面端 / Web 版）。
- 拥有可用的 Anthropic 账号或 API 访问权限。
- 了解基本的 Markdown 语法，便于查看和编辑生成的文件。

## 🚀 快速开始

### 1. 安装

将 `.claude/commands/` 目录复制到你的项目根目录：

```bash
cp -r .claude/commands/ /your/project/.claude/commands/
```

### 2. 创建小说项目

在 Claude Code 中执行：

```
/project:小说-新建
```

按照引导选择：
- 小说类型（言情/玄幻/悬疑/科幻...）
- 篇幅规模（短篇/中篇/长篇/超长篇）
- 叙事视角（第一人称/第三人称限知/全知）
- 写作风格（文艺/爽文/写实/智能适配）

### 3. 设定世界观

```
/project:小说-世界观
```

根据类型选择世界观模板，细化核心设定。

### 4. 创建角色

```
/project:小说-人设
```

创建主角、配角、反派等角色档案。

### 5. 生成大纲

```
/project:小说-大纲
```

选择故事结构，生成分层大纲。

### 6. 开始写作

```
/project:小说-章节
```

按章节撰写正文，系统自动加载所有上下文保持一致性。

## 🎯 核心设计理念

### 借鉴行业标杆

本技能系统借鉴了 [Sudowrite](https://sudowrite.com/) 和 [Novelcrafter](https://novelcrafter.com/) 的核心设计理念：

- **World Bible** — 世界观圣经，确保世界观一致性
- **Character Bible** — 角色圣经，防止角色性格漂移
- **Codex** — 知识库，管理关键设定和术语
- **分层大纲** — 多层级结构，掌控全局节奏
- **伏笔追踪** — Chekhov's Gun 系统，确保伏笔回收
- **状态管理** — 角色状态和时间线追踪，避免逻辑漏洞

### 解决长篇痛点

长篇小说最大的挑战是一致性管理：

- ❌ 角色性格漂移 → ✅ 角色档案实时参考
- ❌ 世界观自相矛盾 → ✅ 世界观圣经约束
- ❌ 伏笔忘记回收 → ✅ 伏笔追踪表自动提醒
- ❌ 时间线混乱 → ✅ 状态管理系统
- ❌ 角色知道不该知道的事 → ✅ 已知信息追踪

## 📝 使用技巧

1. **先规划后写作** — 先用大纲技能规划全局，再逐章写作
2. **及时更新状态** — 每章写完后更新角色状态和伏笔
3. **定期一致性检查** — 每5-10章检查一次逻辑一致性
4. **善用续写技能** — 对不满意的部分用续写技能重写
