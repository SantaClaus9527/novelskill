# 分层大纲生成向导

生成多层级的小说大纲，支持卷→章→场景→节拍的分层结构。

## 前置检查

1. 检查当前目录是否为小说项目（是否存在 `项目信息.md`）
2. 如果不存在，提示用户先执行 `/小说-新建`
3. 读取 `项目信息.md` 获取小说类型和篇幅规模
4. 读取 `世界观.md` 和 `角色/` 目录获取已有设定

---

## 第一步：故事结构选择

**【选项】**
1. 🎭 **三幕式**（开端→对抗→结局，最经典）
2. 🦸 **英雄之旅**（冒险→试炼→归来，适合成长型）
3. 📜 **起承转合**（中国传统叙事，适合言情/历史）
4. 🔀 **多线并行**（多条故事线交织，适合群像/悬疑）
5. 📅 **编年体**（按时间推进，适合历史/史诗）
6. 📝 自定义结构

---

## 第二步：核心故事要素

### 故事核心
**【提示】** 用一句话概括故事核心（主角+目标+冲突）

**【选项】**
1. 用户输入故事核心
2. AI根据已有设定生成候选（可多选）
3. 从一句话简介（项目信息.md）扩展

### 主线冲突
**【选项】**
1. ⚔️ 外部冲突（对抗敌人/势力/环境）
2. 💔 内部冲突（内心挣扎/成长蜕变）
3. 🌍 命运冲突（对抗命运/预言/诅咒）
4. 🔄 多重冲突组合
5. 📝 自定义

### 故事主题
**【选项】**
1. 🌟 成长与蜕变
2. ❤️ 爱情与牺牲
3. ⚖️ 正义与复仇
4. 🔍 真相与谎言
5. 🌈 希望与绝望
6. 📝 自定义

---

## 第三步：分层大纲生成

### 卷/部结构（如适用）
**【选项】**
1. 📚 按故事阶段分卷（开端卷、发展卷、高潮卷、结局卷）
2. 🗺️ 按地理/势力范围分卷
3. ⏰ 按时间段分卷
4. 🚫 不分卷，直接分章
5. 📝 自定义分卷方式

### 章节数量规划
根据篇幅规模自动计算：
- 短篇（5万字内）：10-20章，每章2500-5000字
- 中篇（5-20万字）：20-80章，每章2500-5000字
- 长篇（20-50万字）：80-200章，每章2500-5000字
- 超长篇（50万字+）：200+章，每章2500-5000字

**【选项】**
1. ✅ 使用推荐章节数
2. 📝 自定义章节数

---

## 第四步：场景节拍设计

### 场景粒度
**【选项】**
1. 🎬 粗粒度（每章1-2个主要场景）
2. 🎭 中粒度（每章2-4个场景，推荐）
3. 📍 细粒度（每章4+个场景，适合快节奏）

### 节拍类型
**【选项】**
1. 📈 标准节拍（铺垫→发展→高潮→转折）
2. ⚡ 爽点节拍（打脸→升级→收获，适合网文）
3. 💔 虐心节拍（甜蜜→误会→分离→重逢）
4. 🔍 悬疑节拍（线索→推理→反转→真相）
5. 🔄 混合节拍（根据情节自动调整）

---

## 第五步：高潮与反转设计

### 高潮点规划
**【选项】**
1. 🎯 每卷一个大高潮
2. ⚡ 每10-15章一个小高潮
3. 📈 渐进式高潮（越来越激烈）
4. 📝 自定义高潮节奏

### 反转设计
**【选项】**
1. 🔄 身份反转（某人真实身份暴露）
2. 💔 信任反转（盟友变敌人/敌人变盟友）
3. 🎭 事实反转（真相与表象相反）
4. ⚡ 局势反转（形势突然逆转）
5. 🚫 不设反转
6. 📝 自定义

---

## 第六步：支线设计

**【选项】**
1. ❤️ 感情线（主角的感情发展）
2. 👨‍👩‍👧 家族线（家族恩怨/身世之谜）
3. 🏛️ 势力线（门派/公司/国家斗争）
4. 🔍 悬念线（贯穿全书的谜题）
5. 📈 成长线（能力/心境提升）
6. 📝 自定义支线

**【提示】** 支线应与主线有交汇点，避免完全独立。

---

## 执行动作

1. 创建/更新 `大纲.md` 文件
2. 生成分层大纲结构
3. 标注关键节点（高潮/反转/伏笔埋设点）
4. 更新 `项目信息.md` 中的大纲摘要

### 大纲同步机制（来自项目经验）

**重要提醒**：大纲是活的文档，需要与实际写作保持同步！

**同步时机**：
- 每写完一章，检查大纲是否需要更新
- 当实际剧情偏离大纲时，及时调整大纲
- 当角色状态变化时，更新大纲中的角色信息

**同步内容**：
- 章节标题和核心事件
- 角色状态和发展
- 伏笔埋设和回收
- 时间线推进

**同步方式**：
- 在大纲中标注"已写"或"待写"
- 在大纲中记录实际字数
- 在大纲中记录偏离原因

**【选项】**
1. ✅ 自动同步（AI自动更新大纲）
2. 📝 手动同步（用户手动更新大纲）
3. ⚙️ 部分同步（只同步关键信息）

**大纲.md 结构：**
```markdown
# 故事大纲

## 故事核心
- 一句话概括：
- 主线冲突：
- 核心主题：

## 卷/部结构

### 第一卷：{卷名}
**卷概述：** {简述本卷内容}

#### 第1章：{章节名}
**章节目标：** {本章要达成什么}
**主要场景：**
- 场景1：{地点} — {事件}
- 场景2：{地点} — {事件}
**节拍：**
- 铺垫：{内容}
- 发展：{内容}
- 高潮/转折：{内容}
**伏笔：** {本章埋设的伏笔}
**字数：** {预估}

#### 第2章：{章节名}
...

### 第二卷：{卷名}
...

## 支线规划
### 支线1：{支线名}
- 涉及角色：
- 发展轨迹：
- 与主线交汇点：

## 高潮点标注
- 第X章：{高潮描述}
- 第X章：{高潮描述}

## 反转点标注
- 第X章：{反转描述}

## 伏笔规划
- 第X章埋设 → 第X章回收：{内容}
```

---

## 输出完成信息

1. 大纲结构总览
2. 章节数量和预估字数
3. 关键节点时间线
4. 下一步建议

---

## 工作流指引

### 前置步骤
- `/小说-新建` — 创建项目（必须）
- `/小说-世界观` — 世界观设定（推荐）
- `/小说-角色` — 角色创建（推荐）

### 后续步骤
- `/小说-伏笔` — 设计伏笔系统（推荐）
- `/小说-状态` — 初始化角色状态（推荐）
- `/小说-章节` — 开始写作（最终目标）

### 相关技能
- `/小说-续写` — 扩展/改写已有内容
- `/小说-润色` — 优化文笔
- `/小说-检查` — 一致性检查

### 完整工作流
```
新建 → 世界观 → 角色 → 【大纲】→ 伏笔 → 状态 → 章节写作
```
