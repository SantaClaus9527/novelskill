# 世界观设定向导

构建小说的世界观体系，支持多种类型的世界观模板。

## 前置检查

1. 检查当前目录是否为小说项目（是否存在 `项目信息.md`）
2. 如果不存在，提示用户先执行 `/小说-新建`
3. 读取 `项目信息.md` 获取小说类型，根据类型推荐世界观模板

---

## 第一步：世界观基础框架选择

根据小说类型，提供对应的世界观模板选项：

### 🌸 言情类型
**【选项】**
1. 🏢 现代都市（现实背景，重点在人物关系）
2. 🏰 古代宫廷（架空朝代，宫斗/宅斗元素）
3. 🌍 异国背景（海外留学/跨国恋）
4. 🎭 娱乐圈（明星/经纪人/导演）
5. 🏫 校园青春（高中/大学校园）
6. 📝 自定义（用户输入）

### ⚔️ 玄幻/修仙类型
**【选项】**
1. 🏔️ 传统修仙（炼气→筑基→金丹→元婴→化神...）
2. 🌌 洪荒封神（上古神话体系）
3. 🗡️ 武侠江湖（内力/轻功/门派）
4. 🌐 异世界（穿越重生到新世界）
5. 🎮 游戏异界（游戏系统穿越）
6. 📝 自定义（用户输入）

### 🔍 悬疑/推理类型
**【选项】**
1. 🏙️ 现代都市（刑侦/社会派）
2. 🕰️ 民国背景（旧上海/租界）
3. 🏚️ 暴风雪山庄（封闭空间推理）
4. 🔬 本格推理（纯粹逻辑推理）
5. 👻 灵异悬疑（超自然元素）
6. 📝 自定义（用户输入）

### 🚀 科幻类型
**【选项】**
1. 🌍 近未来（2050年左右的地球）
2. 🚀 太空歌剧（星际文明）
3. 🤖 赛博朋克（高科技低生活）
4. 🧬 生物科幻（基因/进化）
5. ⏰ 时间旅行（平行世界/时间线）
6. 📝 自定义（用户输入）

---

## 第二步：核心设定细化

选择基础框架后，进一步细化核心设定：

### 世界规则设定
**【选项】**
1. ⚡ 力量体系（如有）— 选择或自定义等级划分
2. 🏛️ 社会结构 — 选择政治体制/社会阶层
3. 💰 经济体系 — 选择货币/贸易方式
4. 🌐 地理环境 — 选择或描述主要场景地点
5. 📜 历史背景 — 选择或描述关键历史事件
6. 🔄 全部由AI生成 — 根据类型自动填充

### 世界观规模
**【选项】**
1. 🏠 小型（单一城市/校园/家族，适合言情/校园）
2. 🌆 中型（多个国家/势力，适合都市/武侠）
3. 🌍 大型（完整世界体系，适合玄幻/科幻）
4. 🌌 史诗级（多元宇宙/位面，适合宏大叙事）

---

## 第三步：特色元素添加

**【选项】**
1. ✨ 添加独特设定（用户输入一个独特的世界观元素）
2. 🎭 添加势力/组织（门派/公司/国家/帮派）
3. 📚 添加历史传说（背景故事/神话/预言）
4. 🚫 跳过，使用默认设定

---

## 第四步：世界观检查清单

AI 自动生成世界观检查清单，确保关键要素完整：

**必须项：**
- [ ] 世界名称/背景
- [ ] 核心规则（力量/科技/社会）
- [ ] 主要场景地点
- [ ] 时代背景

**推荐项：**
- [ ] 势力分布
- [ ] 经济/货币
- [ ] 文化/习俗
- [ ] 历史事件
- [ ] 特殊规则/禁忌

如果缺少必须项，提示用户补充。

---

## 执行动作

1. 创建/更新 `世界观.md` 文件
2. 按结构化格式整理世界观内容
3. 生成世界观关系图（文字版）
4. 更新 `项目信息.md` 中的世界观摘要

**世界观.md 结构：**
```markdown
# 世界观设定

## 基础背景
- 世界名称：
- 时代背景：
- 核心规则：

## 地理环境
### 主要场景
- {地点1}：{描述}
- {地点2}：{描述}

## 力量/科技体系
### 等级划分
- {等级1}：{描述}
- {等级2}：{描述}

## 社会结构
### 势力分布
- {势力1}：{描述}
- {势力2}：{描述}

## 经济体系
- 货币：
- 贸易：

## 文化习俗
- {习俗1}：{描述}

## 历史事件
- {事件1}：{时间} - {描述}

## 特殊规则/禁忌
- {规则1}：{描述}
```

---

## 输出完成信息

1. 世界观结构总览
2. 关键设定摘要
3. 下一步建议

## 工作流指引

### 前置步骤
- `/小说-新建` — 创建项目（必须先完成）

### 后续步骤
- `/小说-角色` — 创建角色（推荐）
- `/小说-大纲` — 规划剧情（可选，建议角色创建后执行）

### 相关技能
- `/小说-状态` — 管理角色状态和时间线
- `/小说-伏笔` — 管理伏笔系统

### 完整工作流
```
新建 → 【世界观】→ 角色 → 大纲 → 伏笔 → 状态 → 章节写作
```
