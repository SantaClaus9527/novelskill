# 章节撰写向导

> **别名**：`/小说-写作`、`/小说-创作`
>
> 本技能用于**创建新章节**。如需对已有内容进行续写、扩写、改写，请使用 `/小说-续写`。

按章节撰写小说正文，自动加载项目上下文保持一致性。

## 前置检查

1. 检查当前目录是否为小说项目（是否存在 `项目信息.md`）
2. 如果不存在，提示用户先执行 `/小说-新建`
3. 读取以下文件作为上下文：
   - `项目信息.md` — 基础配置
   - `世界观.md` — 世界观设定
   - `角色/` — 角色档案目录
   - `大纲.md` — 故事大纲
   - `伏笔.md` — 伏笔追踪表
   - `状态.md` — 角色状态
   - `写作风格指南.md` — 风格指南

### 待回收伏笔提醒

```

```

### 伏笔处理选项
**【选项】**
1. ✅ 确认处理所有伏笔任务
2. 📝 选择性处理（指定处理哪些伏笔）
3. ⏭️ 跳过伏笔处理（本章不处理伏笔）
4. ➕ 添加新伏笔（本章需要埋设新伏笔）

**【提示】** 伏笔是小说质量的关键，建议每章至少处理1-2个伏笔。

## 第5.5步：优势伏笔嵌入

> 适用于有"穿越者优势"或"特殊能力"设定的小说。普通小说可跳过此步。

根据世界观设定中的主角优势（如现代科学知识、前世记忆、特殊血脉等），检查本章是否有日常场景可以嵌入"优势伏笔"：

### 嵌入策略
| 类型 | 说明 | 示例 |
|------|------|------|
| 🔍 **观察型** | 主角注意到某个低效做法，内心闪过现代知识但不说出口 | 看到村民粗暴晾晒草药→内心想"低温烘焙才能保留有效成分" |
| 💭 **反射型** | 某个异世界事物触发原世界记忆（感官对比） | 摸到粗糙木桌→想起办公桌上光滑冰冷的金属 |
| 🧮 **分析型** | 主角下意识用现代框架分析眼前情况 | 看到以物易物→"没有一般等价物，交易成本太高了" |
| 🛠️ **动手型** | 主角忍不住做了一个小改进但尽量低调 | 用树枝做了个更省力的杠杆装置 |

### 嵌入原则
1. **不经意感**：优势展示应该是主角的思维习惯，而非刻意炫技
2. **克制为主**：早期章节（前20章）以"观察型"和"反射型"为主，少用"动手型"
3. **不说破**：内心想法不需要用"这就是降维打击"这种话点明，让读者自己品
4. **每章1-2个**：日常场景章至少嵌入1个，纯战斗/剧情推进章可跳过

**【选项】**
1. ✅ 自动嵌入1-2个优势伏笔（根据本章场景智能选择类型）
2. 📝 手动指定嵌入位置和类型
3. ⏭️ 本章不嵌入（纯战斗/高潮/过渡章）

## 第六步：正文生成

AI 根据以上设定生成章节正文：

**生成策略：**
1. 先生成场景框架（每个场景的核心事件）
2. 填充对话和动作
3. 添加环境描写和心理描写
4. 检查角色一致性
5. 嵌入伏笔（如有）

**【选项】**
1. 📄 一次性生成整章
2. 🎬 逐场景生成（每写完一个场景暂停确认）
3. 📝 先写关键对话，再补充描写

---

## 第七步：章节检查

生成完成后自动检查：

### 基础检查
1. ✅ 角色性格是否一致
2. ✅ 世界观是否符合
3. ✅ 与前文是否连贯
4. ✅ 伏笔是否正确埋设/回收
5. ✅ 字数是否达标

### 进阶检查（来自项目经验）
6. ✅ 角色发展速度是否合理（避免过快）
7. ✅ 感情线发展是否自然（避免突兀）
8. ✅ 反派动机是否充分（避免脸谱化）
9. ✅ 伏笔回收是否自然（避免生硬）
10. ✅ 写作节奏是否稳定（避免忽快忽慢）

### 大纲同步检查
11. ✅ 本章内容是否与大纲一致
12. ✅ 如有偏离，是否需要更新大纲
13. ✅ 角色状态是否与大纲预期一致

**检查结果：**
- ✅ 通过 — 无问题
- ⚠️ 警告 — 有小问题但可接受
- ❌ 错误 — 需要修改

---

## 第九步：鉴赏家点评（可选）

章节保存完成后，提示用户：

**【选项】**
1. 🎯 **快速点评** — 调用 `/小说-鉴赏家` 对本章进行快速鉴赏（推荐，只抓亮点和最突出的问题）
2. 🔍 **全面鉴赏** — 调用 `/小说-鉴赏家` 对本章进行全维度深度鉴赏
3. ⏭️ **跳过** — 不鉴赏，直接进入下一章

**【提示】**
- 日常写作建议选「快速点评」，写完5-10章或一卷后再做「全面鉴赏」
- 如果鉴赏报告中发现了系统性问题（非个案，而是技能包本身的盲区），可执行 `/小说-进化` 升级写作技能
- 鉴赏报告中的发现可记录到 `/小说-经验`，沉淀为项目级写作经验

## 工作流指引

### 前置步骤（必须完成）
- `/小说-新建` — 创建项目
- `/小说-世界观` — 世界观设定
- `/小说-角色` — 角色创建
- `/小说-大纲` — 剧情规划
- `/小说-伏笔` — 伏笔设计（可选，但推荐）

### 后续步骤
- 继续 `/小说-章节` — 写下一章
- `/小说-润色` — 优化本章文笔
- `/小说-检查` — 检查一致性
- `/小说-续写` — 扩展/改写本章内容
- `/小说-鉴赏家` — 写完后获得专业点评（步骤九自动触发）

### 相关技能
- `/小说-状态` — 更新角色状态和时间线
- `/小说-伏笔` — 管理伏笔埋设与回收
- `/小说-鉴赏家` — 专业鉴赏点评
- `/小说-经验` — 记录写作经验
- `/小说-进化` — 基于反馈升级技能系统

### 创作循环
```
章节写作 → 鉴赏家点评 → 修改/润色 → 检查 → 状态更新 → 下一章
    ↑                                                  ↓
    └──────────────────────────────────────────────────┘
                                    ↓ (发现系统性问题)
                              /小说-进化 → 技能升级
```
